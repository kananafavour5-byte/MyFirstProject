# Photography Website

## Brief Description of Application
A simple, visually appealing photography website built using HTML and CSS to showcase favorite photos, a biography section, and a contact form.

*Date: 27 January 2026

## Contributors
** Favour **

## Description
This application is a photography portfolio website designed to display a collection of favorite photographs, introduce the creator through a short biography, and allow visitors to get in touch via a contact form.

The purpose of this project is to practice core web development skills including:
* Structure by HTML
* Styling using CSS stylesheet
* Organizing a project using proper file and folder structure
* Using Git and GitHub for version control
* Deploying a live site using GitHub Pages

Simple and clean website, and easy to navigate while still visually appealing.

## Setup/Installation Requirements
1. Download this repository from GitHub.
2. Open the project folder using VS Code.
3. Make sure the following files and folders exist:
        - index.html
        - style.css
        - images folder containing images

No server, database, or external dependencies are required.
